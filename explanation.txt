## Paper: Attention Is All You Need

The main claim of this paper is that sequence-to-sequence tasks can be performed effectively using attention mechanisms alone, without relying on recurrent neural networks such as RNNs, LSTMs, or GRUs. The authors introduce the Transformer architecture, which uses self-attention to capture relationships between words regardless of their distance in a sentence. This design allows the model to process sequences in parallel, reducing training time while improving performance on machine translation tasks.

To evaluate this claim, the paper proposes the Transformer, an encoder-decoder architecture built around multi-head self-attention and feed-forward neural networks. Each encoder and decoder layer contains attention mechanisms, residual connections, and layer normalization. Positional encodings are added to input embeddings so that the model can understand word order. Unlike recurrent models, the Transformer processes all tokens simultaneously, making it more efficient and better at handling long-range dependencies.

The model is evaluated on machine translation benchmarks, primarily the WMT 2014 English-German and English-French datasets. Performance is measured using the BLEU score, which compares machine-generated translations with human reference translations. The Transformer is compared against previous state-of-the-art sequence models based on RNNs, LSTMs, and GRUs. The experimental results show that the Transformer achieves higher BLEU scores while requiring significantly less training time, demonstrating the effectiveness of attention-based architectures for sequence modeling.
